# OIBSIP Graphics Design Tasks

## Overview

This repository contains the graphic design tasks completed as part of the Oasis Infobyte Internship Program (OIBSIP). All designs were created using Canva and demonstrate skills in branding, visual communication, layout design, and creative content development.

## Tools Used

* Canva
* Typography Principles
* Color Theory
* Graphic Design Fundamentals

## Tasks Completed

### Task 1: Posters & Flyers

Designed two promotional posters:

* TechNova 2026 Technical Festival Poster
* Save Our Planet Awareness Campaign Poster

### Task 2: Logo Design

Created a minimalist logo for a fictional brand named PixelCraft Design Studio.

Deliverables:

* Full Color Logo
* Monochrome Logo
* Reversed Logo

### Task 3: Business Card Design

Designed a professional double-sided business card for PixelCraft Design Studio.

### Task 4: Infographic

Created an infographic presenting the Benefits of Drinking Water using icons, visual elements, and structured information.

### Task 5: Tri-Fold Brochure

Designed a tri-fold brochure for PixelCraft Design Studio featuring:

* About Us
* Services
* Recent Projects
* Contact Information
* Brand Highlights
